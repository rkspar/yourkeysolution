HTML STRUCTURE

2.1. Basic Structure

The Basic Structure contains:

<div id="wrapper" class="wide-wrap">

<header class="header-container header-type-classic"></header> - socials sharing, main navigation(responsive)

<div class="heading-container"></div>- page title, breadcrumb section

<div class="content-container"></div>- all the content
<div class="footer-widget"></div>- footer contact information

<footer id="footer" class="footer"></footer> - footer info, footer navigation, footer-copyright...

</div>

STYLESHEETS

3.1. Files

All CSS are located in the "css" folder of the template. Each CSS File is well commented and easy to use and customize to your own needs. Here's the list of the CSS files:

  1. style.css – Main style of theme.
  2. font-awesome.min.css – Font Awesome 4.2.4 – The iconic font and CSS toolkit.
  3. elegant-icon.css – Elegant font toolkit.
  4. section.css – Sections style.
  5. form-style.css – Form Styles.
  6. settings.css – Revolution Slider Styles.
  7. shop.css – Shop page style settings.
  8. preloader.css – Preloader Style.
  9. magnific-popup.css – Magnific Popup library.

Note: style.css, font-awesome.min.css, preloader.css must declare on all pages of Yolia.
